Collaborators: Kylynn Fuhriman, Casey Wolf, Kaleb Fulwider, Lindsey Glod
Creation Date: 09/11/2025

## Description
Collaborative project to create UVSim used to interpret BasicML, a signed 4-digit representing a word
UVSim has a 100 word memory, CPU, and memory registers

## Installation
pip install pytest

## Modules
import pytest<br>
import sys

## Instruction for use

To run the program, utilize the terminal or command line with the following: python filename.py file.txt
File.txt must only contain words defined as "a signed four-digit decimal number, such as +1234, -5678"
In the event you are prompted for input by the program, it must be a signed/unsigned integer number
Any failure for file.txt or input will be met with an exception/error and exiting the program
